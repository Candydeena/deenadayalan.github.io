My Projects

Eye Gesture Communication for people of motor disabilities

This project enables people with motor disabilities to communicate using eye gestures. It uses OpenCV and MediaPipe to detect eye gaze direction and blink patterns, which are mapped to control commands. The system provides voice feedback and can control external devices like Arduino-driven motors or wheelchairs, allowing hands-free communication and movement assistance.

Real-Time Financial Anomaly Detection for financial detection

This project detects unusual or suspicious financial transactions in real time. It employs machine learning algorithms such as Random Forest and Isolation Forest to identify anomalies based on transaction patterns. The system improves fraud detection efficiency by analyzing live transaction data and flagging irregular behaviors instantly.
